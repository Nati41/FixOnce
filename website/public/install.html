Install via Terminal

For developers

Quick setup if you prefer the command line. Requires manual MCP configuration afterward.

# macOS / Linux
# Requires: Python 3.10 or newer

# 1. Navigate to FixOnce directory
cd ~/Desktop/FixOnce    # or wherever you extracted it

# 2. Verify Python version (must be 3.10+)
python3 --version

# 3. Check requirements.txt exists
ls requirements.txt || echo "ERROR: requirements.txt not found!"

# 4. Create virtual environment
python3 -m venv venv
source venv/bin/activate

# 5. Install dependencies
pip install -r requirements.txt

# 6. Start server (will auto-select port 5000-5009)
python src/server.py --flask-only

# Server will print the actual port it's running on.
# Dashboard: http://localhost:PORT
#
# After server is running, configure MCP manually
# or paste the AI Setup prompt into your AI assistant.
