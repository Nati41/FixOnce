Frequently asked questions

What is Memory Review?
As you work, AI proposes memories — decisions, solutions, rules. These appear in your dashboard for review. You can approve, edit, or reject each one. Only approved memories persist across sessions.
How is this different from ChatGPT memory?
ChatGPT memory is a black box — you don't see or control what it remembers. FixOnce gives you a dashboard to review, edit, and approve every memory. You decide what sticks.
Is my code uploaded anywhere?
No. FixOnce runs entirely on your machine. Project memory is stored in a .fixonce/ folder inside your project directory. Nothing is sent to any server.
Which AI tools are supported?
FixOnce works with Claude Code, Cursor, Codex (via MCP), and Windsurf. Any tool that supports MCP (Model Context Protocol) can connect to FixOnce.
Does it work on Windows?
Yes. The Windows installer is fully tested. It installs FixOnce, sets up the background service, and configures MCP integration for supported tools.
Does it work on Mac?
Yes. Both Intel and Apple Silicon Macs are supported. The installer handles everything — server setup, auto-start configuration, and tool integration.
Why do I need a Git or project folder?
FixOnce stores memory relative to your project. Opening from an empty folder or the home directory won't work — you need a real project context for memory to be meaningful.
What happens if I uninstall?
The uninstaller removes FixOnce from your system. Project memory in .fixonce/ folders is preserved unless you delete it manually — so you can reinstall and pick up where you left off.
What happens when multiple developers or AI agents work on the same project?
Today, FixOnce focuses on helping individual developers maintain reliable project memory. Our long-term vision is larger: a shared memory layer where developers and AI agents can learn from previous decisions, solved bugs, project rules, and accumulated knowledge — instead of starting from zero. Team memory and shared project knowledge are among the most important directions we're actively exploring.
Is this free?
Yes, during the beta. We're focused on getting the core experience right. Pricing for future versions hasn't been decided yet.
