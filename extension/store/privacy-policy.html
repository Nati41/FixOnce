FixOnce Privacy Policy

Last updated: February 2026

Summary: FixOnce captures JavaScript errors from websites YOU choose to monitor. All data stays on YOUR local machine or YOUR server. We never collect, store, or transmit any data to external servers.

What Data We Collect

When you enable FixOnce on a website, the extension captures:

  • JavaScript errors - Error type, message, and stack trace
  • HTTP errors - Failed network requests (4xx/5xx status codes)
  • Source location - File name and line number where the error occurred
  • Timestamp - When the error happened

What We Do NOT Collect

  • Personal information (names, emails, addresses)
  • Passwords or authentication tokens
  • Form input data
  • Cookies or session data
  • Browsing history
  • Any data from websites where FixOnce is not explicitly enabled

Where Data Is Stored

All captured error data is sent exclusively to:

  • Your local FixOnce server (default: localhost:5000)
  • Your self-hosted FixOnce instance (if configured)
Important: FixOnce does NOT send any data to our servers or any third-party services. You have complete control over your data.

How Data Is Used

The captured error data is used solely to:

  • Display errors in your local FixOnce dashboard
  • Help AI assistants (like Claude) understand and fix bugs in your code
  • Build a searchable history of past solutions

User Control

You have full control over FixOnce:

  • Opt-in only - FixOnce is inactive by default. You must explicitly enable it for each website.
  • Whitelist management - You can enable/disable tracking for any site at any time.
  • Data deletion - You can clear all captured data from your local FixOnce server at any time.
  • Complete removal - Uninstalling the extension removes all extension-related data.

Permissions Explained

  • storage - To save your whitelist preferences locally
  • activeTab - To detect the current website domain
  • tabs - To show domain info in the popup
  • host_permissions (all_urls) - To capture errors on websites you choose to monitor. This permission is only used on sites you explicitly whitelist.

Data Security

  • All data transmission occurs over your local network
  • No data is ever sent to external servers
  • Your error data never leaves your machine unless you configure a remote FixOnce server

Children's Privacy

FixOnce is a developer tool not intended for use by children under 13. We do not knowingly collect any information from children.

Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated revision date.
