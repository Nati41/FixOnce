Privacy Policy

Last updated: March 2026

Summary: FixOnce stores all data locally on your machine. We do not collect, transmit, or store any data on external servers. You have complete control over your data.

What FixOnce Stores

FixOnce stores the following data locally on your machine:

  • Project decisions - Architectural choices and their reasons
  • Insights - What was learned during development
  • Solutions - How errors were fixed
  • Session context - Current goals and work area

Where Data Is Stored

  • ~/.fixonce/ - User-specific data (your machine only)
  • .fixonce/ - Project-specific data (travels with your project)
Important: All data stays on YOUR machine. FixOnce does NOT send any data to external servers.

Browser Extension (Optional)

If you install the optional browser extension:

  • It captures JavaScript errors from websites you choose to monitor
  • Errors are sent to your local FixOnce server (localhost only)
  • Sensitive data is automatically sanitized (emails, tokens, API keys)
  • You control which sites are monitored via whitelist

AI Integration

Note: FixOnce shares your project context with AI assistants (Claude, Cursor, Codex) that you use. This is the core feature - enabling AI to remember your decisions and context.

The AI assistant sees:

  • Project decisions and insights you've saved
  • Current goals and work context
  • Error history (if extension is installed)

This data flows from your local FixOnce server to the AI tool running on your machine.

What We Do NOT Collect

  • No analytics or usage tracking
  • No telemetry
  • No personal information
  • No data sent to FixOnce servers (we don't have any)
  • No third-party services receive your data

Data Control

You have full control:

  • View all data - Check ~/.fixonce/ and .fixonce/ folders
  • Delete anytime - Remove ~/.fixonce/ to clear all user data
  • Export - Your data is in JSON format, fully portable
  • No account required - FixOnce works entirely offline

Updates

FixOnce does not auto-update. You control when to update via:

  • Manual git pull
  • AI-assisted update (copy prompt)
